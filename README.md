# ANRO TRIP — Путешествия и командировки

Современный веб-сайт туристического агентства **ANRO TRIP** (г. Екатеринбург). Более 18 лет опыта в организации путешествий и командировок по всему миру. Консьерж-сервис с полным циклом обслуживания.

Собран на **Astro 5** и **Tailwind CSS 4** с фокусом на производительность, Core Web Vitals и accessibility.

---

## Особенности

- **Адаптивный дизайн** — корректная работа на десктопе и мобильных
- **Поиск туров** — виджет поиска (авиабилеты / туры и отели) через Tourvisor и Nemo
- **Bento-сетка** — популярные направления, блок «Почему выбирают»
- **Lightbox** — просмотр отзывов-документов с прокруткой и зумом
- **Модальные окна** — обратный звонок, подарок, форма отзыва
- **View Transitions** — плавные переходы между страницами
- **Производительность** — оптимизация под Core Web Vitals, LCP, AVIF для Hero
- **Accessibility** — skip link, ARIA-labels, семантический HTML, WCAG 2.2
- **Glass-панели** — backdrop-blur (Header, SearchWidget) с учётом production-багов
