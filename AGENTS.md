# ANRO TRIP — Сайт туристического агентства

Веб-сайт туристического агентства «ANRO TRIP» (г. Екатеринбург). Более 18 лет опыта в организации путешествий и командировок по всему миру.

## Технологический стек

- **Astro 5.x** — основной фреймворк с Server-Side Rendering
- **Tailwind CSS 4.x** — утилитарный CSS-фреймворк с кастомными дизайн-токенами
- **TypeScript** — строгая типизация
- **Sharp** — оптимизация изображений
- **View Transitions API** — плавные переходы между страницами
- **astro-iconify** — иконки

## Структура проекта

```
src/
├── assets/           # Изображения, шрифты, медиафайлы
├── components/       # Astro компоненты
│   ├── ui/          # Базовые UI компоненты (Modal)
│   ├── widgets/      # Виджеты поиска (Nemo, Tourvisor)
│   └── *.astro      # Секции страниц
├── layouts/          # Макеты страниц
├── pages/           # Маршруты
│   ├── index.astro  # Главная страница
│   ├── corp.astro   # Корпоративным клиентам
│   ├── cabinet.astro # Личный кабинет
│   └── *.astro      # Дополнительные страницы
├── styles/          # Глобальные стили
└── types/           # TypeScript определения типов
```

- `glass-panel` — полупрозрачная панель с blur-эффектом

## Соглашения по коду

### Компоненты Astro

- Все компоненты используют TypeScript интерфейсы для Props
- Props с русскими комментариями для документации
- Дефолтные значения для опциональных пропсов

### CSS / Tailwind

- Использовать `@theme` для определения дизайн-токенов
- Применять `clamp()` для fluid typography
- Mobile-first: классы для мобильных — без префикса, для десктопа — `md:`, `lg:`
- Accessibility: skip-link, aria-labels, семантический HTML

### Изображения

- Формат WebP для растровой графики
- AVIF для Hero-изображений
- Предварительная загрузка критичных изображений через `<link rel="preload">`

### Accessibility

- Skip link для клавиатурной навигации
- ARIA-labels на интерактивных элементах
- Семантические теги (header, main, footer, nav, article)
- Поддержка скринридеров

## Особенности реализации

### View Transitions

- Используется `ClientRouter` из `astro:transitions`
- Исключение для Tourvisor: полная перезагрузка страницы при переходе на главную
- Анимации через CSS `::view-transition-*`

### Модальные окна

- Модалки: CallbackModal, GiftModal, ReviewModal
- Блокировка скролла с компенсацией ширины скроллбара
- Функции `lockScroll()` / `unlockScroll()`

### Поиск туров

- Виджеты NemoSearch и TourvisorSearch
- Интегрируются через скрипты виджетов
- Особое поведение на главной странице

## Рекомендации по модификации

### Добавление новой страницы

1. Создать файл в `src/pages/`
2. Импортировать Layout
3. Добавить Header и Footer
4. Использовать существующие компоненты секций

### Добавление новой секции

1. Создать компонент в `src/components/`
2. Следовать паттерну: Props интерфейс → деструктуризация → HTML шаблон
3. Добавить стили в `global.css` через `@layer components`

### Работа с изображениями

- Добавлять в соответствующую папку `src/assets/`
- Оптимизировать через Sharp перед коммитом
- Использовать WebP формат
