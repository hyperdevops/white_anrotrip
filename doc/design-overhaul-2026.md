**Основа:** изучение lookup.design, Tailwind CSS, Material Design 3, Apple HIG, тренды веб-дизайна 2026 (Contented), Uprock, LandingFolio; конституция проекта и текущее руководство.

## 1. Сводка трендов из всех источников

### 1.1 lookup.design
- **Hero:** один сильный заголовок, одна CTA, иллюстрация или градиент, минимум текста, крупная типографика.
- **Карточки:** тень + скругления, внутренняя иерархия (заголовок → контент), одна CTA внутри.
- **Searchbar:** поиск как главный элемент, иконка лупы + очистка (×), теги/пилюли под полем.
- **Баннеры:** одна мысль, контраст текста к фону, одна кнопка, бейджи «NEW» при необходимости.
- **Footer:** тёмный фон, 4–5 колонок с заголовками, соцсети, один CTA (подписка/контент), копирайт внизу.
- **Navigation:** логотип слева, ссылки по центру/справа, одна выделенная CTA справа, минимализм.
- **Теги/чипы:** пилюли категорий под поиском для быстрого переключения (как направления).

### 1.2 Tailwind CSS
- Стили только утилитами, без лишнего CSS; JIT для произвольных значений.
- Showcase (NASA, Shopify, Vercel и др.) — уровень качества интерфейсов.
- Готовые блоки Tailwind Plus — референс для Hero, карточек, форм.

### 1.3 Material Design 3
- **Цвет:** палитра primary/secondary, контраст, темизация.
- **Типографика:** шкала размеров, иерархия, читаемость.
- **Motion:** Ripple, переходы (shared axis, fade), быстрая обратная связь.
- **Состояния:** hover, focus, active, disabled — явно заданы для интерактивных элементов.

### 1.4 Apple HIG (Liquid Glass)
- **Materials:** полупрозрачность, размытие, слои — «стекло» без перегруза.
- Чёткая иерархия, минимум визуального шума, аккуратное использование глубины.

### 1.5 Тренды веб-дизайна 2026 (Contented)
- **Минимализм с эмоциями:** мягкая анимация, типографика как акцент, реакция на действия.
- **Цвет и градиенты:** уход от «безопасных» палитр; тёплые и холодные вместе; градиенты с глубиной/шумом.
- **Микровзаимодействия:** кнопки, загрузка, реакция форм — интерфейс «общается».
- **Типографика:** контрастные пары шрифтов, нестандартные заголовки, крупный текст; читаемость в приоритете.
- **Сетки:** асимметрия, смещение блоков при сохранении логичной структуры.
- **Правило:** не перегружать; один тренд — качественно; тренд под проект и бренд.

### 1.6 Uprock
- Header/Footer: лучшие практики, что включать и чего избегать.
- Глассморфизм: характеристики, принципы.
- Дизайн-система: 10 обязательных компонентов, опора на документацию (Material, Spectrum).

### 1.7 LandingFolio
- Референсы лендингов по секциям (Hero, карточки, баннеры, футер).
- Компоненты под Tailwind — идеи структуры и стиля без копирования кода.

## 3. Переработка по разделам

### 3.1 Цветовая система

**Принцип:** бренд не зафиксирован — можно выбрать **любую** трендовую палитру. Цвета ниже приведены по документам (Contented 2026, lookup.design, Material Design 3, Apple HIG, LandingFolio) как примеры; тон и оттенки можно менять под проект.

**Роли цветов (M3, lookup):**
- **Primary** — основной акцент: ссылки, иконки, выделение, навигация.
- **Secondary** — второй акцент: подзаголовки, второстепенные кнопки, теги.
- **Accent / CTA** — одна яркая кнопка (Hero, ключевые действия); контраст с фоном обязателен (WCAG).
- **Surface** — фон контента (часто белый или очень светлый нейтральный).
- **Footer** — тёмный тон для контраста с основным контентом (lookup, Campaignmonitor).

**Общие правила (из документов):**
- **Контраст:** текст на кнопках и фонах — по WCAG (минимум 4.5:1 для текста, 3:1 для крупного).  
- **Один главный акцент:** не смешивать 3+ ярких цвета в одном блоке (Contented, lookup).  
- **Градиенты:** допустимы с глубиной, лёгким шумом; не перегружать (Contented).  
- **Реализация:** все цвета задавать в `@theme` в `global.css` (--color-primary, --color-secondary, --color-cta, --color-cta-hover, --color-footer-bg и т.д.) и использовать через Tailwind; при необходимости JIT: `bg-[#hex]`.

**Выбор палитры:** привязки к бренду нет — можно взять любой вариант A–D, смешать оттенки из разных вариантов или задать свою палитру по тем же ролям (primary, secondary, CTA, surface, footer), опираясь на тренды из документов.

### 3.2 Типографика

**Цель:** сложная типографика 2026 при сохранении читаемости; контраст пар (Montserrat + Inter).

| Элемент | Рекомендация | Источник |
|--------|---------------|----------|
| Базовый размер | Оставить fluid: `clamp(14px, 0.875rem + 0.25vw, 16px)` в `html`. | Текущий гайд, Contented |
| h1–h6 | Оставить clamp-шкалу в `global.css`; Montserrat для заголовков. | Текущий гайд |
| Секционные подзаголовки | Ввести единый паттерн: маленький капслок над заголовком (например «ПОПУЛЯРНЫЕ НАПРАВЛЕНИЯ» / «ЧТО МЫ ПРЕДЛАГАЕМ») — как на lookup «WHAT WE OFFER». Цвет: primary или gray-600. | lookup |
| Тело текста | Inter, достаточный line-height; в длинных блоках — `text-balance` где уместно. | M3, Contented |
| Кнопки и метки | Montserrat, bold/extra-bold, uppercase где уже используется. | Текущий гайд |

**Действия:**
- Добавить в дизайн-систему опциональный «секционный лейбл»: классы типа `text-xs font-bold uppercase tracking-widest text-primary` или `text-gray-600` для подзаголовка над h2.
- Не менять базовые h1–h6 в `global.css` без необходимости; все кастомизации — через утилиты в компонентах.

### 3.12 Сетки и отступы

**Цель:** единый ритм, при желании — лёгкая асимметрия (Contented).

| Элемент | Рекомендация | Источник |
|--------|---------------|----------|
| Контейнер контента | `max-w-7xl mx-auto px-4 sm:px-6` везде. | Текущий гайд |
| Секции | `space-y-0` между блоками; вертикальные отступы внутри секций. | index.astro |
| PopularTours | Bento + Container Queries — оставить. | Гайд, конституция |
| Смещение блоков | Допустимо лёгкое смещение колонок или карточек для «движения»; не ломать читаемость. | Contented |

**Действия:**
- Проверить все секции на единый `px-4 sm:px-6` и max-w-7xl.
- Не вводить новые значения отступов без причины (например, придерживаться 4/6/8/12/16/24).

### 3.13 Z-Index и модальные окна

**Цель:** не менять иерархию (гайд); модалки поверх всего.

| Уровень | Значение | Использование |
|---------|----------|----------------|
| Модалки | z-[2147483647] | GiftModal, CallbackModal, ReviewModal, оверлеи Reviews |
| ScrollToTop | z-[2147483646] | Кнопка «Наверх» |
| CookieBanner | z-[2147483640] | Нижний баннер |
| OfficeWidget, FavoritesWidget | z-[2147483630] | Плавающие виджеты |
| ScrollProgress | z-9999 | Полоса прогресса |
| Header | z-50 | Шапка |

| Hero | z-0 | Первый экран |

**Действия:**
- Не менять порядок; при добавлении новых плавающих элементов — вписать в эту таблицу и использовать арбитражные значения в скобках: `z-[...]`.
